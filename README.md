## Como executar um arquivo lua
```powershell
lua filename.lua
```

## Instalação do Lua
Scoop
```powershell
iwr -useb get.scoop.sh | iex
```

Extras
```powershell
scoop bucket add extras
```

Lua
```powershell
scoop install lua
```
